## 🖥️ Installation

```bash
# Frontend Setup
git clone https://github.com/gfranco7/SpringbootFinal
npm install
npm start

# Backend Setup (Spring Boot)
# Configure application.properties with your DB credentials
mvn spring-boot:run

```
